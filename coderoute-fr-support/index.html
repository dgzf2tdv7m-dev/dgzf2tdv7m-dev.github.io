Questions frequentes

Comment acheter CodeRoute FR ?

L'achat se fait dans l'App Store. Si une option payante est disponible dans l'app, elle est geree par votre compte Apple.

L'app fonctionne-t-elle hors ligne ?

Oui. Les questions, les corrections et les panneaux deja inclus dans l'application restent accessibles sans connexion Internet.

Ou les donnees sont-elles stockees ?

Les resultats, les erreurs et la progression sont stockes localement sur votre appareil. CodeRoute FR ne cree pas de compte utilisateur et ne collecte pas de donnees personnelles sensibles.
